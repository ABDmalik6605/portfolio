## ğŸ¨ Customization

### Update Personal Information

1. **Hero Section** (`components/Hero.tsx`):
   - Update your name and titles
   - Modify the description

2. **About Section** (`components/About.tsx`):
   - Update your bio and stats
   - Modify highlights

3. **Education** (`components/Education.tsx`):
   - Add your educational background
   - Include certifications

4. **Projects** (`components/Projects.tsx`):
   - Showcase your projects
   - Update project images and links

5. **Skills** (`components/Skills.tsx`):
   - List your technical skills
   - Adjust proficiency levels

6. **Contact** (`components/Contact.tsx`):
   - Update contact information
   - Modify social media links

### Color Scheme

Edit `tailwind.config.ts` to change the color scheme:
```typescript
colors: {
  primary: { /* your colors */ },
  accent: { /* your colors */ },
}
```
